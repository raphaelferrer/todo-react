# ✅ Todo App - React + TypeScript

Aplicação de lista de tarefas desenvolvida em React com TypeScript, utilizando Hooks, organização de componentes e boas práticas de front-end.

Projeto criado como parte do curso:

> 🎓 Criando um App com React, TypeScript, Hooks, Context e Forms

---

## 🚀 Tecnologias utilizadas

- React
- TypeScript
- Create React App
- UIKit (UI)
- Animate.css
- LocalStorage

---

## 📚 O que foi praticado

- Componentização com React
- Tipagem com TypeScript
- Props e estado
- Eventos e formulários
- Persistência no LocalStorage
- Animações básicas
- Organização de projeto
